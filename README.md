# EWAT
My modules written for [https://github.com/toxic-ig](toxic-ig)'s [https://github.com/toxic-ig/ewat](EWAT project).
## Modules
The only module available right now is Spoof.
### Spoof
Usage:
```python
from modules import spoof

email = spoof.Email()
```
or use it as a standalone:
```bash
cd modules
python spoof.py
```
